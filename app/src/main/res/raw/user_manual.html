1. Introduction

Welcome to SensorBot, an intuitive application designed to guide you in choosing between two distinct chatbot conditions: Pervasive and Reference. This manual provides a step-by-step guide to navigate the app, ensuring a smooth and pleasant experience.



2. Choosing a Chatbot Condition

On the main screen:

  • Two radio buttons labeled "Pervasive" and "Reference" are presented.
  • Select either the "Pervasive" or "Reference" radio button by tapping on your desired choice.

Note: Only one option can be selected at a time.



3. Accepting Terms and Conditions

  • Beneath the radio buttons, you'll find a checkbox labeled "By clicking this you are accepting our terms and conditions".
  • To view the full terms and conditions, click on the "terms and conditions" hyperlink. A modal dialog will appear, displaying the details.
  • To proceed, make sure to check the checkbox indicating your acceptance.


4. Accessing and Navigating the Chat Activity

Upon tapping the "Next" button on the main screen, you'll enter the Chat Activity. This interactive environment facilitates a two-way communication between you and the chatbot.


4.1. Asking Questions

  • User Interface: A chat window is displayed where you can see both your questions and the chatbot's responses.
  • Interactivity: To ask a question, simply type it into the input field and tap the "Send" or "Submit" button. The chatbot will then analyze your query and provide an answer.
  • Supported Topics: The chatbot is equipped to provide information on:
    • Computational Offloading
    • Sensor Data Distribution
    • Network Sharing
    • File Sharing or Transfer
    • Finding Missing Sensor

4.2. Chatbot Condition: Pervasive vs. Reference

  • Pervasive Condition: In this mode, the chatbot actively assists you. Upon asking about any of the supported topics, the chatbot doesn't just provide information, but also takes proactive steps to help you. For instance, if you ask about "Finding missing sensor" , the chatbot might start an automatic sensor detection process.
  • Reference Condition: Under this condition, the chatbot acts as a reference guide. If you ask about "File Sharing or Transfer", it will provide a step-by-step guide on how to do so, but the actual implementation is up to you.

4.3. Tips for Effective Communication

  • Be Specific: For best results, ask specific questions like "How do I initiate file sharing?" rather than general ones like "Tell me about file sharing."
  • Rephrase: If the chatbot doesn't understand your query the first time, try rephrasing it or breaking it down into simpler terms.
